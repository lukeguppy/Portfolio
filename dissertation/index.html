Final Year Dissertation (1st-Class)

My dissertation focused on developing an intelligent autonomous vehicle system using advanced machine learning techniques to navigate complex traffic environments safely and efficiently.

Key Features:

  • PPO Reinforcement Learning Agent: Implemented a Proximal Policy Optimization algorithm to train an AI agent for real-time decision-making in dynamic traffic scenarios.
  • Custom Unity Simulation: Created a realistic 3D environment simulating traffic lights, intersections, and multiple vehicles for thorough testing and training.
  • Extendable Reward System: Designed a modular framework allowing easy addition of new behaviours and objectives for the agent, enabling adaptable learning.
  • Technologies Used: C#, Unity Game Engine, PPO Deep Reinforcement Learning.
